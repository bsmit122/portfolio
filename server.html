Server and Networking

At home I run my own server which is based on TrueNAS Scale, an open-source Linux operating system. Maintaining the server has taught me networking, Docker, and other useful computer skills.

Functionality

  • Self-hosted cloud storage
  • Automated backup of all my devices
  • HomeAssistant smart-home hub
  • Secure password management
  • Network-attached 3D printing

Methods

  • SSH and basic terminal usage
  • Docker container deployment
  • PFSense router
  • ZFS file storage array

Results

  • The server saves me over $15 per month compared to similar services like Google Drive, Backblaze, and LastPass, while drawing only 30W of power.
